Corrector Ortográfico — Proyecto en C
Este programa lee un archivo de texto, construye un diccionario, contabiliza la frecuencia de cada palabra y sugiere correcciones basadas en variantes generadas de la palabra ingresada.
Cómo compilar
gcc main.cpp -o corrector
Cómo ejecutar
./corrector
Archivos incluidos
	•	main.cpp — Código fuente del programa.
	•	diccionario.txt — Archivo base con palabras.
	•	.gitignore — Archivos que Git no debe rastrear.
